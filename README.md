# 🛡️ LLM без тормозов: анализ галлюцинаций и Guardrail на основе NER

> Проект по исследованию неконтролируемой генерации и отсутствия критического мышления у LLM с разработкой легковесного Guardrail для детекции галлюцинаций.

## 📖 Описание проекта

Проект посвящен исследованию феномена **галлюцинаций** в больших языковых моделях (LLM) — ситуации, когда модель генерирует уверенные, но полностью вымышленные ответы. 

В рамках работы:
1. Проведен анализ энтропии и стабильности ответов LLM на вымышленные и реальные вопросы
2. Разработан легковесный **Guardrail** на основе NER (Named Entity Recognition) для детекции и блокировки галлюцинаций
3. Выполнено тестирование на датасете из 25 промптов (75 генераций)

**Основной стек:** GigaChat (LLM), spaCy (NER), Python, pandas, matplotlib

---

## 🎯 Актуальность

Проблема галлюцинаций LLM — **один из главных барьеров** для внедрения чат-ботов в реальный бизнес:

| Сфера | Риск галлюцинаций |
|-------|-------------------|
| Банкинг | Неверные суммы, даты, обещания списания кредитов |
| Медицина | Опасные советы по лекарствам |
| E-commerce | Выдуманные скидки и характеристики товаров |
| Юриспруденция | Несуществующие законы и прецеденты |

Данный проект предлагает **простое, быстрое и интерпретируемое решение** для детекции ~60% типичных галлюцинаций без дообучения модели.

## 🎯 Цели и задачи

### Цель
Исследовать поведение LLM при генерации вымышленных фактов и разработать легковесный Guardrail для защиты от галлюцинаций.

### Задачи

| № | Задача |
|---|--------|
| 1 | Создать датасет из 25 промптов (20 вымышленных + 5 реальных)
| 2 | Реализовать генерацию ответов GigaChat с разными температурами (0.1, 0.5, 0.9)
| 3 | Провести анализ энтропии через множественную генерацию
| 4 | Разработать Guardrail на основе NER (spaCy)
| 5 | Оценить эффективность Guardrail (Precision, Recall, F1)
| 6 | Проанализировать ошибки и предложить улучшения
